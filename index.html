Process

進め方

  1. 01
    界隈リサーチ

    色・フォント・モチーフ・雰囲気を界隈ごとに言語化。トンマナをルールとして整理する。

  2. 02
    Figma設計

    デザインシステム・コンポーネント化。カラートークン・タイポスタイルを定義してから組む。

  3. 03
    AI実装

    Claude Code / MCP で高速コーディング。「なぜそうしたか」を説明できる形で納品する。

  4. 04
    公開・自動化

    Vercel デプロイ。n8n・API連携で運用コストを下げる仕組みも提案できます。
